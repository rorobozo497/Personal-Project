HTML Basics

HTML (HyperText Markup Language) is the backbone of any website. It provides the structure and content of a webpage. All websites are made from coding, so although you may use websites to create your own landing page without coding, technically all of that tedious process is done by the website writing the code to help suit what styles or structures you use for your landing page.

  • Tags: HTML uses tags to define elements, such as headings, paragraphs, and links. In the example below, you can see the beginning words of ever line to be in tags. (The greater and less than sign for you beginners). You can NOT start a line with out these tags for any define elements.
  • Attributes: Tags can have different styles, such as font style or color, to add extra information, like id or class.
  • Structure: A basic HTML document has a <head> (for style of the website itself) and a <body> (this is the part where you see the words you are reading right now).

Example:

                <!DOCTYPE html>
                <html>
                  <head>
                    <title>My First Page</title>
                  </head>
                  <body>
                    <h1>Hello, World!</h1>
                    <p>This is a paragraph.</p>
                  </body>
                </html>

CSS Basics

CSS (Cascading Style Sheets) controls the appearance and layout of a webpage. It allows you to make your website visually appealing.

  • Selectors: Select elements to style (e.g., body, #id, or .class). This changes the style such as color and structure of the sections.
  • Properties: Define what to style, like color, font size, or margin.
  • Values: Set the value for each property, like red, 16px, or auto.

Example:

            body {
                background-color: lightblue;
                font-family: Arial, sans-serif;
            }
           
            h1 {
                color: darkblue;
                text-align: center;
            }

JavaScript Basics

JavaScript is a programming language that adds interactivity and logic to your website. It's used for dynamic content like animations, form validation, and games.

  • Variables: Stores data (e.g., let name = 'Alice';). This ensures that the code knows what you are talking about if you define that variable.
  • Functions: Create reusable blocks of code (e.g., function greet() { alert('Hello!'); }).
  • Events: Add interactivity, like clicks or keypresses (e.g., button.addEventListener('click', myFunction);). This is what increase the score counter everytime you press the square in the game above.

Example:

                document.addEventListener('DOMContentLoaded', () => {
                    const btn = document.getElementById('myButton');
                    btn.addEventListener('click', () => {
                        alert('You clicked the button!');
                    });
                });
